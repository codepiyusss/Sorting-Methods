# Sorting Algorithm Analyzer

## Overview

This is a simple Python project that demonstrates different sorting algorithms. The user can enter an array, choose a sorting method, and view the sorted result along with its execution time and complexity.

The project was created for learning and comparing the behavior of various sorting techniques.

## Features

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Execution time measurement
* Time and space complexity display
* Comparison of all sorting algorithms
* Array statistics (size, minimum, maximum, average)
* Save sorting results to a text file

## Example

Input:

5 2 8 1 9

Output:

Sorted Array: [1, 2, 5, 8, 9]

Time Complexity: Best: O(n), Average: O(n²), Worst: O(n²)

Execution Time: 0.00001234 seconds
